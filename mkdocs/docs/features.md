- A modern responsive web app that works on any resolution.
- You can search the prompt library for any keywords.
- Video tutorials showing how to use the prompt library and its integrations.
- Complete history of the prompt's evolution since the moment it was submitted by the original
  author.
- You can bookmark the prompts that you like.
- You can export the bookmarked prompts using different formats. The supported ones are:
    - **Epam AI Dial:** An integration that allows you to import the prompts to Epam AI Dial right
      away.
    - **Zeus:** custom-built format that includes all the prompt's assessment information &
      evolution
      history.
- You can download each prompt separately as a .json file.
